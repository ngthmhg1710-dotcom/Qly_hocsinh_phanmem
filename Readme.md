# EDU Ver.0 - Student Management & Teaching Support System

Phần mềm hỗ trợ giáo viên quản lý học sinh, điểm danh, nhập điểm, theo dõi kết quả học tập và tổ chức các hoạt động hỗ trợ giảng dạy.

## Chức năng chính

### Quản lý học sinh

* Quản lý danh sách lớp học
* Thêm, sửa, xóa học sinh
* Import danh sách học sinh từ Excel
* Quản lý hồ sơ và ghi chú học sinh

### Quản lý chuyên cần

* Điểm danh thủ công
* Điểm danh bằng QR Code
* Thống kê chuyên cần

### Quản lý kết quả học tập

* Nhập điểm học sinh
* Theo dõi kết quả học tập
* Lưu lịch sử chỉnh sửa điểm
* Xuất báo cáo Excel/PDF

### Công cụ hỗ trợ giảng dạy

* Gọi tên ngẫu nhiên
* Quiz nhanh
* Bảng điện tử mini
* Đồng hồ đếm ngược
* Ghi chú nhanh

### Trò chơi học tập

* Flashcard
* Nghe chọn hình
* Sắp xếp câu
* Vòng quay may mắn

## Công nghệ sử dụng

* C#
* Windows Forms
* SQLite
* Excel/PDF Libraries
* Google Drive API

## Môi trường chạy

* Windows 10/11
* .NET Framework
* PC hoặc Laptop
